# Dubz HUD

A modern DarkRP HUD and UI suite for Garry's Mod.

Dubz HUD replaces most of the default DarkRP interface with a cleaner, more consistent experience while staying lightweight and easy to configure. It's built on DubzFramework and is designed to be something server owners can install, customize, and forget about.

## Features

* Fully custom HUD
* Animated status bars
* Custom ammo display
* Door interface
* Scoreboard
* MOTD
* Death screen
* Death notices
* Voice panel
* Weapon selector
* Vehicle HUD
* Notification system
* Overhead player info
* Laws menu
* Voting interface
* Connection UI
* Config menu
* Multi-language support

## Installation

Drop both folders into your Garry's Mod addons directory:

```
garrysmod/addons/dubzframework
garrysmod/addons/dhud
```

Restart the server or reload Lua.

## Configuration

Open the configuration panel:

```
dubzhud
```

or

```
dhud_config
```

The master Dubz configuration panel can be opened with:

```
dubzconfig
```

Server configuration is saved inside:

```
garrysmod/data/dhud/
```

## Requirements

* Garry's Mod
* DarkRP
* DubzFramework

## Philosophy

I built this because I wanted a HUD that looked modern without trying too hard. The focus is smooth animations, consistent styling, good performance, and making everything configurable instead of hardcoded.

This project is actively maintained and will continue receiving updates as DubzFramework grows.
